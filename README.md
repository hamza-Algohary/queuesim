# Queue System Simulator
This is a program applying the concepts of queueing theory.

Built using Kotlin, Python, Gtk4 and libadwaita.

# Building
```bash
git clone https://github.com/hamza-Algohary/queuesim
cd queuesim
./gradlew run
```
